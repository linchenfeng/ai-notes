## 复盘
- 今天学到了什么：
  - 状态码是 API 语义的一部分
  - `raise HTTPException` 比返回 error dict 更规范
  - 资源不存在要用 404，不该返回 200
- 还没弄懂什么：
  - 更完整的全局异常处理方式
  - 如何统一错误响应模型并覆盖全部异常场景
- 明天开始前要准备什么：
  - 学习异常处理中间件与全局 handler
  - 尝试补一套统一错误响应设计
