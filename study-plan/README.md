# 90 天学习路线索引

- 起始日期：2026-03-25
- 结束日期：2026-06-22

## 阶段目标
- 第 1 阶段：先跑通最小闭环（Day 01～Day 14）：先会 Python 最小开发、会调模型 API、会做一个最简单的 AI 工具。
- 第 2 阶段：先做出第一个全栈小项目（Day 15～Day 28）：尽快把你的前端优势接进来，做一个有界面的 AI 应用。
- 第 3 阶段：进入最重要的能力 RAG（Day 29～Day 49）：学会做知识库问答，这是初学者最值得投入的主线。
- 第 4 阶段：评测与优化（Day 50～Day 63）：从能跑升级到更准、更稳。
- 第 5 阶段：工具调用和数据库接入（Day 64～Day 77）：让模型不只是会说，还会调用工具。
- 第 6 阶段：Agent、流程编排、部署（Day 78～Day 87）：在已有能力上做复杂一点，但不追求很炫，追求稳定。
- 第 7 阶段：作品集和面试准备（Day 88～Day 90）：把学过变成可展示、可面试、可投递。
