# Day 10 参数实验记录

## 使用模型
qwen2.5:0.5b

## 测试任务
请用中文写一段适合放在项目 README 里的自我介绍，控制在 80 字以内。

## 实验结果

### 实验 1：低随机性 + 较短输出
- temperature: 0.2
- max_tokens: 40
- 输出结果：

欢迎加入我们的团队！我们致力于为用户提供高质量的软件解决方案。如果您有任何问题或需求，请随时与我们联系。

### 实验 2：中等随机性 + 中等输出
- temperature: 0.7
- max_tokens: 80
- 输出结果：

```markdown
# 细节

### 名称：[你的名字或团队名称]
#### 电子邮件地址：[你的邮箱地址]
#### Twitter/LinkedIn: [你自己的社交媒体账号]

如果您有任何问题或需要帮助，请随时联系我。
```
请注意，这个示例中的姓名、电子邮件和Twitter/LinkedIn是虚构的，并且在实际情况中应替换为实际的信息。

### 实验 3：高随机性 + 更长输出
- temperature: 1.0
- max_tokens: 120
- 输出结果：

这是一个关于我的项目的简单简介。

## 我的观察
- 低 temperature 的输出通常更稳，更接近常规答案。
- 高 temperature 的输出通常更活跃，但也更容易发散。
- max_tokens 越大，模型通常越有空间生成更长的内容。
