# Day 36：本地索引环境

## 目录作用

这个目录用于承接 RAG 项目里和“本地索引”相关的文件。

它的目标不是今天就完成全部索引构建，而是先把后面会持续写入的内容位置约定好，让 Day37、Day38、Day39 的产出有明确落点。

## 当前采用的轻量方案

当前先采用：

- 本地文件索引
- 本地 JSON 配置
- 后续脚本逐步写入 chunk、embedding 和检索结果

## 当前目录说明

- `index_config.json`
记录当前索引环境配置和文件路径约定

- `chunks.json`
后续可保存切分后的 chunk 列表

- `embeddings.json`
后续可保存 chunk 对应的向量结果

- `search_results.json`
后续调试检索器时可保存样例输出

## 一句话总结

这个目录就是当前 RAG 小项目的本地索引工作区。
