# Day 19~20 联调与体验清单

## Day 19：前后端联调
- 前端页面可打开：`frontend/day17_app.html`
- 后端服务可启动：`uvicorn backend.day18_main:app --reload --port 8000`
- 点击“生成总结”后可收到后端 JSON
- 页面结果区可显示模型输出

## Day 20：基础 UX 优化
- loading：点击生成后按钮切换为“生成中...”
- 错误提示：请求失败时在结果区显示红色错误信息
- 复制按钮：生成结果后可一键复制
- 成功反馈：复制成功后显示“已复制到剪贴板”
- mock 开关：可切换 mock 数据，便于页面调试

## 验收标准
- 主流程可走通：输入 -> 点击 -> 展示结果
- 至少一种风格稳定可用（`brief`）
- 错误和成功状态都可见
