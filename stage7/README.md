# 第 7 阶段：作品集和面试准备

- Day 范围：88～90
- 阶段目标：把“学过”变成“可展示、可面试、可投递”。

## 阶段日程
- [Day 88](day88.md)：整理 GitHub。
  任务：每个项目补 README、架构图、运行说明、截图。
  产出：规范项目仓库。
- [Day 89](day89.md)：整理简历项目描述。
  任务：每个项目写清楚项目背景、技术栈、你的职责、遇到的问题、如何优化。
  产出：简历项目文案。
- [Day 90](day90.md)：模拟面试。
  任务：准备并口述这些问题：什么是 RAG、Embedding 有什么作用、如何减少幻觉、工具调用怎么设计、你项目里做过哪些优化。
  产出：面试题答案 + 90 天总结。
