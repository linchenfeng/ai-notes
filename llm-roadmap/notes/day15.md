## 我现在理解了什么
- Prompt 不是随便问一句话，而是任务说明书
- 一个好 Prompt 通常要包含角色、任务、约束、输出格式
- system 更适合放长期规则
- user 更适合放本轮具体任务
- Prompt 写得越清楚，模型输出通常越稳定
